# TikTok Auto Unfollow - Расширение для Chrome

Автоматическая массовая отписка от пользователей в TikTok с настраиваемыми параметрами. Расширение позволяет отписываться как от обычных подписок, так и от взаимных подписок («Друзья») с задержкой 2 секунды между действиями для защиты от ограничений TikTok.

---

## 🚀 Особенности

### ✨ Умное распознавание кнопок
- Автоматически находит и считает кнопки **«Following»** и **«Friends»** по отдельности
- Динамически сканирует страницу и фильтрует кнопки по видимости и тексту
- Работает с динамической DOM-структурой TikTok

### 🔄 Автопрокрутка
- Автоматически прокручивает список подписок для загрузки всех аккаунтов
- До 50 итераций прокрутки с умным определением конца списка
- Позволяет отписаться от сотен аккаунтов за один сеанс

### 📊 Отслеживание прогресса в реальном времени
- Отображает текущее количество отписанных аккаунтов
- Отдельные счётчики для кнопок **«Following»** и **«Friends»**
- Показывает общее количество отсканированных и отфильтрованных кнопок
- Чёткие сообщения о статусе на каждом этапе

### ⚙️ Гибкие настройки
- Выберите, отписываться ли от обычных подписок, от взаимных подписок («Друзья») или от обоих типов
- Флажки для выбора типов кнопок для нажатия
- Для запуска необходимо выбрать хотя бы один вариант

### ⏱️ Защита от ограничений
- Задержка 2 секунды между каждым действием отписки
- Помогает избежать ограничений и блокировок со стороны TikTok
- Безопасно для вашего аккаунта

## 📦 Установка и настройка

1. **Скачайте или клонируйте** это расширение на свой компьютер
2. Откройте Chrome и перейдите по адресу `chrome://extensions/`
3. Включите **«Режим разработчика»** (переключатель в правом верхнем углу)
4. Нажмите **«Загрузить распакованное расширение»** и выберите папку с расширением

---

## 📖 Как использовать

### Пошаговая инструкция

1. **Войдите в TikTok**
   - Откройте [tiktok.com](https://www.tiktok.com) в браузере
   - Войдите в свой аккаунт TikTok

2. **Перейдите в свой профиль**
   - Нажмите на иконку профиля в правом верхнем углу

3. **Нажмите на «Подписки» (Following)**
   - В своём профиле перейдите на вкладку **«Подписки»**
   - Появится список всех аккаунтов, на которые вы подписаны

4. **Прокрутите страницу вниз для загрузки кнопок**
   - Прокрутите страницу, чтобы стали видны кнопки «Following»
   - Убедитесь, что на странице загружены хотя бы несколько кнопок

5. **Нажмите на иконку расширения**
   - Нажмите на иконку расширения на панели инструментов Chrome
   - Откроется всплывающее окно расширения

6. **Выберите параметры отписки**
   - Отметьте **«Unfollow Following»** для отписки от обычных подписок (включено по умолчанию)
   - Отметьте **«Unfollow Friends»** для отписки от взаимных подписок (опционально)
   - Вы должны выбрать хотя бы один вариант

7. **Нажмите «Start Auto Unfollow»**
   - Расширение начнёт процесс:
     - Отобразит «Loading all followers...» во время прокрутки списка
     - Покажет «Found: X followers» с разбивкой по типам
     - Начнёт отписку с отображением прогресса в реальном времени
     - По завершении покажет «Done — Following: X | Friends: Y»
   - Между каждым действием установлена задержка 2 секунды
   - Кнопка блокируется во время работы и разблокируется по завершении

## 📊 Понимание интерфейса

### Сообщения о статусе

| Сообщение | Описание |
|-----------|----------|
| **«Loading all followers...»** | Расширение прокручивает страницу для загрузки всех подписчиков |
| **«Found: X followers»** | Всего найдено подписчиков, от которых можно отписаться |
| **«Following: X \| Friends: Y»** | Текущее количество отписанных в реальном времени |
| **«Done — Following: X \| Friends: Y»** | Итоговый подсчёт по завершении |

### Раздел «Details» (Подробности)

Отображает три элемента информации:
- **Scanned** — Всего элементов кнопок найдено на странице
- **Filtered** — Всего кнопок, соответствующих критериям «Following» или «Friends»
- **Breakdown** — Количество найденных пользователей каждого типа

## ⚠️ Важные замечания

### Перед началом работы
- Расширение работает только при авторизации в TikTok
- Убедитесь, что страница полностью загружена перед нажатием «Start»
- Откройте вкладку **«Подписки»** (не любую другую вкладку)

### Во время работы
- Не покидайте страницу во время отписки
- Держите браузер открытым для лучших результатов
- Задержка 2 секунды предотвращает блокировку аккаунта
- Большие партии могут занять 30+ минут в зависимости от количества

### Рекомендации
- Начинайте с небольших партий для тестирования (20-50 пользователей за раз)
- Делайте перерывы между длительными сессиями отписки
- Следите за уведомлениями TikTok и предупреждениями об ограничениях
- Если аккаунт заблокирован, подождите несколько часов перед повторной попыткой

## 🔧 Устранение неполадок

**Кнопки не найдены?**
- Убедитесь, что вы находитесь на вкладке «Подписки»
- Дождитесь полной загрузки страницы
- Попробуйте сначала прокрутить страницу вручную

**Расширение не работает?**
- Перезагрузите расширение на странице `chrome://extensions/`
- Убедитесь, что TikTok полностью загружен
- Попробуйте закрыть и снова открыть всплывающее окно

**Низкая производительность?**
- Обработка большого количества подписчиков может занять время
- Задержка 2 секунды установлена намеренно для защиты от ограничений
- Закройте другие вкладки для снижения нагрузки на процессор
